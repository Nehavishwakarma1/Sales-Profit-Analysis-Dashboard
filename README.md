📊 Sales & Profit Dashboard

🔹 Project Overview : 
The Sales & Profit Dashboard is a data analytics project built using Microsoft Power BI to analyze business sales performance and profitability.
This dashboard provides interactive visualizations to understand sales trends, profit distribution, and regional performance.

🔹 Objectives : 
Analyze overall sales and profit performance
Track monthly sales trends
Compare performance across regions and states
Identify top-performing categories and sub-categories
Understand the impact of discounts on profit

🔹 Key Features : 
📌 Interactive filters for region, category, and segment
📊 KPI cards for Sales, Profit, and Quantity
📈 Monthly sales and profit trend analysis
🌍 State-wise and region-wise performance
🛒 Category and sub-category insights
💡 Discount vs Profit analysis

🔹 Tools & Technologies : 
Microsoft Power BI
Data Cleaning & Transformation
Data Modeling
DAX (Data Analysis Expressions)

🔹 Key Insights : 
Identified top-performing regions and products
Observed monthly sales and profit trends
Found impact of discounts on profitability
Highlighted high and low-performing categories

🔹 Outcome : 
This project demonstrates the ability to transform raw sales data into meaningful insights using interactive dashboards, helping businesses make data-driven decisions.
